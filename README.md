# cad-birdhouse
This is a birdhouse I created on OnShape! It is meant to be simple, affordable, feature a birdbath, and it has a large ring for easy hanging! I decided to create this birdhouse so I can use it in my own home and I also generally thought this would be a cool project to make. I also created a poster and research slides to assist in creation of this project!
